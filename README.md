# 개인정보 흐름표 자동화 프로젝트

이 프로젝트는 기업 및 조직 내 개인정보의 수집, 처리, 저장, 공유 및 삭제 과정을 시각적으로 표현하는 개인정보 흐름도를 자동으로 생성하는 도구입니다. 사용자는 입력된 정보를 바탕으로 데이터 흐름을 시각화하여 개인정보 처리 프로세스를 명확히 이해하고 관리할 수 있습니다.

---

## 주요 기능

### 1. 개인정보 처리 단계 시각화
- **수집**:
  - 입력된 개인정보 수집 채널 및 데이터 항목 표시.
- **저장**:
  - 저장 위치 및 암호화 여부 표시.
- **공유 및 제공**:
  - 외부로 공유되는 데이터 및 대상 표시.
- **삭제**:
  - 데이터 삭제 시점 및 절차 표시.

### 2. 사용자 입력 기반 흐름 생성
- 간단한 입력 양식을 통해 개인정보 흐름 데이터를 입력.
- 데이터 처리 과정의 관계를 자동으로 매핑.

### 3. 시각화 도구
- **Flowchart.js**, **Mermaid.js**와 같은 라이브러리를 활용한 시각화 지원.
- 흐름표를 PNG, PDF, SVG 형식으로 내보내기 가능.

### 4. 보고서 생성
- 시각화된 흐름표와 함께 프로세스 설명이 포함된 보고서를 자동 생성.
- Markdown, Word, Excel 등의 형식으로 출력 가능.
